🎵 UVR Online – Separador de Áudio com Inteligência Artificial

📖 Descrição

O UVR Online é um sistema desktop desenvolvido em Python para processamento de áudio utilizando Inteligência Artificial. A aplicação permite separar automaticamente voz e instrumental de músicas utilizando o Demucs, além de oferecer recursos de alteração de tom, velocidade e masterização inteligente.

O sistema possui uma interface moderna desenvolvida em Tkinter, autenticação por login, integração com MySQL para registrar os processamentos realizados e acompanhamento do progresso em tempo real.

✨ Funcionalidades

- 🔐 Sistema de login.
- 🎵 Separação de voz e instrumental com IA (Demucs).
- 📂 Seleção de arquivos de áudio.
- 📊 Barra de progresso do processamento.
- 🎛️ Alteração de tom (Pitch).
- ⏩ Alteração da velocidade do áudio.
- 🎚️ Masterização inteligente utilizando Pedalboard.
- 💾 Registro do histórico de processamentos no banco de dados MySQL.
- 📁 Organização automática dos arquivos modificados e masterizados.
- ⚠️ Tratamento de erros e mensagens para o usuário.

🛠️ Tecnologias Utilizadas

- 🐍 Python
- 🖥️ Tkinter
- 🗄️ MySQL
- 🤖 Demucs
- 🎧 FFmpeg
- 🎚️ Pedalboard
- 🔢 NumPy
- 🧵 Threading

📂 Estrutura do Projeto

- Interface gráfica moderna.
- Banco de dados MySQL.
- Separação de áudio por IA.
- Processamento em segundo plano.
- Pasta Modificados para áudios editados.
- Pasta Masterizados para áudios masterizados.

🎯 Objetivo

Desenvolver uma ferramenta intuitiva para separação, edição e masterização de áudio utilizando Inteligência Artificial, proporcionando rapidez, qualidade e facilidade de uso.
